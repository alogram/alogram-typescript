# Changelog

All notable changes to the Alogram PayRisk TypeScript SDK will be documented in this file.

## [0.1.0] - 2026-02-09

### Added
- Initial release of the PayRisk TypeScript SDK.
- Fetch-based client for browser and Node.js environments.
- Support for synchronous and asynchronous signal ingestion.
- Built-in retry logic and security guardrails.
- Full TypeScript definitions.
